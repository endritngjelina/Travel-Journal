# 🌍 Travel Journal

A personal travel journal web project built with HTML, showcasing travel experiences,
destinations, and memories across multiple journal entries.

## 📖 Overview

This project is a multi-page travel journal created with pure HTML. Each page
represents a unique travel entry, documenting destinations, highlights, and 
personal reflections from various journeys.
